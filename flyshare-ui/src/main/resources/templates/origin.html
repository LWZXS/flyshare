工厂级溯源

Factory-Level Traceability

工厂级溯源是指，对生产企业进行审核及现场验核，对来源于该生产企业的商品从生产阶段到出厂、流通等环节的原产地、 品质和中文标识等信息进行采集、验证，并出具溯源证书的过程。

注册账号

Registration

请点击“全球质量溯源服务平台”右上方“注册”功能，进行用户注册。

企业备案

Enterprise Filing

溯源申请主体（企业）须在“全球质量溯源服务平台”完成备案。

备注：一个注册账号对应一个备案企业。

溯源申请

Traceability Application

1) 验厂申请

已完成备案的企业可在“全球质量溯源服务平台”发起验厂申请。

2) 溯源商品登记

完成验厂的生产商，可在“全球质量溯源服务平台”进行溯源商品登记操作。

3) 溯源授权管理

生产商完成商品登记后，可在“全球质量溯源服务平台”将其下属商品（商品登记号识别商品唯一性）授权给品牌商或贸易商，以使得被授权的企业可以申请该商品的工厂级溯源证书，未获得授权的企业则不能申请该商品的工厂级溯源证书。

4) 生产批次申报

生产商在“全球质量溯源服务平台”进行生产批次申报操作。

5) 工厂级溯源证书申请

具有商品权限的生产商或者品牌商、贸易商等溯源申请主体（企业）可在“全球质量溯源服务平台”申请工厂级溯源证书，提交申请后，给予证书申请企业申请回执。

溯源申报

Traceability Declaration

溯源申报企业须按照“全球质量溯源服务平台”的申报规范进行溯源申报；对于办理了工厂级溯源的商品，根据证书申请回执，填写相关字段关联溯源证书。

溯源查询

Traceability Query

办理过工厂级溯源业务的商品可通过溯源码、电子交易订单号、物流单号等条件查询商品的溯源信息。
