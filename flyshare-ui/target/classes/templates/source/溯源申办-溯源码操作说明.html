溯源码操作说明

溯源码申请

Traceabilty code application

已完成溯源备案的企业可登陆“全球质量溯源服务平台”申领溯源码。

溯源码分为实物码与电子码两类。

溯源码领取

Traceabilty code requisition

当企业申请实物码时，直接派送溯源码（码卷）给到企业；当企业申请电子码时，企业需安装溯源码取码工具，根据需要从“全球质量溯源服务平台”获取。

绑码操作环节说明

Instructions of coding

工厂级

目前工厂级溯源的赋码操作环节主要有三个节点可供选择，分别是生产时赋码、发货时赋码、口岸赋码 有条件的生产商，在生产过程中引入赋码工序，并在批次号申报时，提交绑码关系。 申请溯源的商品发货时，有条件的工厂或贸易商，可以按照商品种类分别赋码，并在申请工厂级溯源证书时，提交绑码关系。 口岸赋码因业务不同而操作不同，若是一般贸易业务，则可在码头进行赋码操作，并登陆“全球质量溯源服务平台”提交绑码关系，也可根据报检信息先绑定电子码，获取后打印码图在中文标签上；若是跨境电商保税进口，则可在仓库发货时赋码在商品或订单上，并通过出仓报文提交绑码关系。

贸易商级

目前贸易商级溯源的赋码操作环节主要有两个节点，分别是发货时赋码、口岸赋码（同工厂级溯源）。
